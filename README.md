# Init Next.js

```
npm init
npm install react@latest next@latest react-dom@latest
```

```
  "scripts": {
    "dev": "next dev"
  },
```

```
npm run dev
```

# Install Tailwindcss

```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

# Install Daisy UI

```
npm i -D daisyui@latest
```

```
module.exports = {
  //...
  plugins: [
    require('daisyui'),
  ],
}
```
